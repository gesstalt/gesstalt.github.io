Objectives

  • To understand computer programming and its roles in problem solving
  • To understand and develop well-structured programs using C language
  • To learn the basic data structures through implementing in C language

Learning Outcomes

  • Problem solving through computer programming
  • Familiarity of programming environment in Linux operating system
  • Ability to use different memory allocation methods
  • Ability to deal with different input/output methods
  • Ability to use different data structures

Course content

  • Introduction to digital computers, Number systems - binary, octal, hexa, and conversion between the number systems, binary arithmetic
  • Introduction to programming, Problem solving and expression of solution through flow chart and algorithm
  • Parts of a program - primitive data types, variables, operators and their precedence, expressions, input/output, conditionals and branching, looping statements
  • Functions, Storage classes - scope and life time, recursion
  • Arrays, Pointers, User defined data types - structures, unions, Dynamic allocation, File Handling, Linear data structures List, Stack, and Queue, Time and space requirements

Text Books

  • T3. Herbert Schildt; C: the complete reference; McGraw-Hill Osborne Media.

Reference Books

  • R2. Thomas L. Floyd, Digital Fundamentals, Prentice Hall.
