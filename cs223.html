Objectives

  • To understand the best practics in software engineering
  • To develop necessary skills to handle software projects in a principled way

Course Outcomes

  • Ability to analyze and specify software requirements.
  • Ability to apply software engineering principles
  • Ability to develop scalable software systems.
  • Ability to plan and work effectively in a team

Course content

  • Introduction: Problem of software development, problem of scale, basic process approach.
  • Software Process Models: Concept of processes, process specification, process models and utilities.
  • Advanced object modeling: Structural and behavioral diagrams using UML.
  • Requirement analysis and specification: Structured analysis, Function points, validation.
  • Design principles: Partitioning, top-down, bottom-up, refinement, coupling, cohesion, design on UML.
  • Coding: Style, structured programming, verification concepts.
  • Testing: Types of testing, test case generation, test plan, test scenario, regression testing.
  • Project planning: Effort, schedule, quality, monitoring, configuration management.
  • Agile software development: process model, project management, SCRUM, SPRINT.
